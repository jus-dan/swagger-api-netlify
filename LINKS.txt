
Technologie-Stack:
🚀 Frontend: React + Vite
🔧 Backend: Node.js + Express + Netlify Functions
🗄️ Datenbank: Supabase (PostgreSQL)
📚 API-Dokumentation: Swagger/OpenAPI
☁️ Deployment: Netlify


Verfügbare Features:
📋 Personen anzeigen - GET /person
➕ Neue Person erstellen - POST /person
🔍 Einzelne Person abrufen - GET /person/{id}
👋 Einfache Begrüßung - GET /hello
📖 Interaktive API-Dokumentation - Swagger UI
