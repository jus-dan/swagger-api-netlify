
Technologie-Stack:
🚀 Frontend: React + Vite
🔧 Backend: Node.js + Express + Netlify Functions
🗄️ Datenbank: Supabase (PostgreSQL)
📚 API-Dokumentation: Swagger/OpenAPI
☁️ Deployment: Netlify


Verfügbare Features:
🔐 Authentifizierung & Rollen-Management
📋 Personen anzeigen - GET /person (mit Berechtigungen)
➕ Neue Person erstellen - POST /person (mit Berechtigungen)
🔍 Einzelne Person abrufen - GET /person/{id} (mit Berechtigungen)
👋 Einfache Begrüßung - GET /hello
📖 Interaktive API-Dokumentation - Swagger UI
⚙️ Admin-API für Rollen- und Berechtigungsverwaltung
